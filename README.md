# Netflix SQL Data Analysis

## Overview
SQL analysis project exploring Netflix content trends and business insights using PostgreSQL.

## Tools Used
- SQL
- PostgreSQL
- pgAdmin

## Analysis Performed
- Data cleaning and validation
- Duplicate checks
- Aggregations
- Joins
- Subqueries
- CASE statements
- Ranking functions
- Trend analysis
- Content distribution insights

## Dataset
Netflix Movies and TV Shows dataset

## Key Outcomes
Derived insights into content growth, genre distribution, production trends, and platform content patterns.
